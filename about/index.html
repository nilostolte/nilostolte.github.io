NILO STOLTE GRAPHICS

About

Nilo Stolte completed his PhD in computer graphics at Paul Sabatier University, Toulouse, France, in 1996. His research dealt about ray-tracing, 3D visualization, voxelization, robust voxelization, octree, 3D rendering, implicit surfaces, point rendering and interval arithmetic.

Prior to his Masters' degree, he had considerable experience in embedded, concurrent and Assembly programming (particularly for the 8085 processor) as well as several high level languages. During his Masters' his main interests were in computer graphics and computer architectures, among other areas in computer science. He started programming in C language and also used some 8086 Assembly.

During his post-doc at SUNY Stony Brook and Duke University, he continued his research in computer graphics and started reasearch in parallel processing, proposing a new dynamic load balancing algorithm for multiprocessing memory shared machines. He also started programming in C++.

He then switched from an academic position of Senior Lecturer in London to a C++ Software Engineer position in a computer graphics company in Toulouse, France, where he dealt with multi-spectral ray-tracing for military applications.

After this experience, he switched back to an academic position of Assistant Professor in Singapore and of "Professeur Agrégé" in Montreal, Canada. He acquired Canadian citizenship and started a career in Java Android development in Motorola, Samsung, and Sony Ericson. This experience allowed him to have a very good perception of the problem of the use of frameworks for GUI development, starting with Java Android and now in Java desktop.

He has been nowadays studying and developping several tools for developping GUIs using vector graphics, as a way to obtain portable and multiplatform GUIs. He studied, used and modified several vector graphics open source packages (FreeHEP, PDFBox and Opentype.js) for transforming Java vector to other vector formats, to generate PDF and to use vector fonts and vector typography on desktops and enbedded applications as well as to automatically generate formatted texts in PDF. His interest has been also extended to ways to have web applications using the same code used in desktop and embedded applications.
